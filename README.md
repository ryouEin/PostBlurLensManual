# PostBlurLensマニュアル

## 導入〜加工までの流れ

1. UnityにてアバターにPostBlurLensを導入
1. VRChatにてPostBlurLensを使用して写真を撮影
1. 撮影した写真をWEBアプリにて加工（https://pbl.doll-profile.com/）
